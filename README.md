
# Real Estate Portfolio Project 🏠

A modern, high-end, and fully responsive real estate landing page designed for seamless property discovery and management. This project features a professional UI/UX, smooth scroll animations, and interactive elements.

---

## ✨ Key Features

* **📱 Fully Responsive Design:** Optimized for all screen sizes (Mobile, Tablet, Desktop) using Bootstrap 5.
* **🔍 Advanced Search Functionality:** Integrated search filters for Location, Property Type, and Price Range.
* **🎭 Smooth Animations:** Powered by **AOS (Animate On Scroll)** for a dynamic browsing experience.
* **🎠 Interactive Testimonials:** A touch-enabled customer review slider using **Swiper.js**.
* **🏢 Property Showcase:** Beautifully designed cards to display property details like beds, baths, and square footage.
* **📲 App Integration:** Dedicated section for Google Play and App Store links to encourage mobile app downloads.
* **❓ Dynamic FAQ:** Functional accordion-style Frequently Asked Questions section.
* **📧 Newsletter Subscription:** Styled footer section with a newsletter signup form.

---

## 🛠️ Built With

The project leverages modern web technologies and libraries:

* **Framework:** [Bootstrap v5.2](https://getbootstrap.com/)
* **Animations:** [AOS (Animate On Scroll)](https://michalsnik.github.io/aos/)
* **Slider:** [Swiper JS](https://swiperjs.com/)
* **Icons:** Font Awesome 6.5.1 & Bootstrap Icons
* **Typography:** [Google Fonts (Inter)](https://fonts.google.com/specimen/Inter)

## 📝 Usage Notes

  * **🌐 CDN Dependency:** This project requires an active internet connection to load **AOS**, **Swiper**, and **Google Fonts** CDNs.
  * **🎨 Customization:** You can easily change the primary branding color by modifying the `.custom-blue` and `.bg-primary` classes in the CSS.
  * **🖼️ Images:** Ensure your `images/` directory contains the correct filenames as referenced in the HTML (e.g., `house.jpg`, `freepik_0001.png`).
